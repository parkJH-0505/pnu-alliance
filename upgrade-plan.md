# PNU Alliance 웹사이트 고도화 계획

## 뿌리문서 핵심 인사이트

### 정체성
- 호스트: 박준홍 (직접 큐레이션, 콜드컨택, 기록)
- "서울에서 일하는 부산대 출신들이 서로의 성장에 실질적으로 기여하는 신뢰 기반 네트워크"
- 기존 동문회와 완전히 다름 — 형식적 모임이 아닌 "좋은 사람들이 자연스럽게 연결되고, 정보와 기회와 영감이 순환하는 생태계"
- 지방 거점국립대 출신의 구조적 불균형(연결의 부재, 정보의 비대칭, 기회의 편중)을 깨는 것이 출발점
- "부산대학교라는 공통 배경이 초기 신뢰를 담보"

### 우주 탐사 세계관 (커리어 라운드)
- Ground Crew: 발사 전 연료를 채우는 시간 (재학생, 취준생)
- Launcher: 첫 점화, 궤도에 오르려는 중 (사회 초년생)
- Rocket: 본격적인 궤도 돌파 (30대 실무 리더)
- Orbiter: 안정적인 비행 궤도 진입 (팀장~본부장급)
- Galaxy: 은하의 중심, 사람을 연결하고 영감을 줌 (C-level)
- Cosmos: 산업 생태계를 설계하고 세대를 잇는 존재 (명예 단계)
- "위계가 아닌 서로 다른 궤도를 비행하는 탐사자"

### 핵심 문화/철학
- 여정의 존중 — 지금의 스펙이 아니라 방향과 태도
- 질문하는 문화 — "말하지 마. 들어."
- 심리적 안전 — Chatham House Rule
- 선순환 — 받기만 하는 관계는 지속되지 않음
- 금기: 허세, 정치, 성비 목적, 무임승차, 자기PR 과잉, 영업

### 핵심 명언들 (브랜딩 활용)
- "사람이 깡통이다" — 이대한
- "부산대학교로 신뢰 담보를 깔고 가는 거" — 박재혁
- "처음에 눈덩이를 뭉치면 시럽고 힘들다. 그것만 잘하면 커진다" — 강경록
- "모으는 건 어렵지 않다. 유지하는 게 어렵다" — 노주현
- "지금은 스카이가 롤모델이지만, 나중에는 후배님이 만든 게 롤모델이 되는 거예요" — 이대한
- "고생은 이제 저까지만 하고 후배들부터는 조금 편하게 했으면" — 이대한
- "부산대 출신이 서울에서 혼자 싸우지 않아도 되는 세상" — 박준홍

## 고도화 방향 (사용성 높고, 유지관리 적은 품)

### 텍스트/컨셉 고도화
1. 모든 카피를 뿌리문서의 철학과 톤에 맞게 전면 재작성
2. "우주 탐사 여정" 세계관을 Tiers 섹션에 더 풍부하게 반영
3. 선배들의 명언을 Testimonial 밴드에 활용
4. "부산대 출신이 서울에서 혼자 싸우지 않아도 되는 세상" 비전 강조

### 새로운 섹션/기능 추가 (유지관리 최소)
1. **Manifesto 섹션** — PNU Alliance의 철학과 가치관을 선언문 형태로 (한 번 작성하면 끝)
2. **Journey Timeline** — 2025.09부터의 실제 여정을 시각적 타임라인으로 (투명성/진정성)
3. **Culture Code 섹션** — 행동 규칙과 금기를 세련되게 표현 (멤버들이 자부심 느끼는 문화)
4. **Host 소개 섹션** — 박준홍 호스트의 진정성 있는 소개

### 제거/통합할 것
- 별도 탭/페이지 분리 없이 원페이지 유지
- 게시판 형태의 지속 업데이트 필요 기능은 만들지 않음
- Events/News는 현재 형태 유지 (정적 데이터, 필요시 코드에서 직접 수정)
